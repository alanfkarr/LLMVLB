This GitHub directory contains datasets generated for the paper ``Large Language Model Variability Laid Bare'' by Alan F. Karr, Manuhat Akkala and Regina Ruane. 
The paper has been submitted to ASA Discoveries. 
Copies may be requested by e-mail to alan.karr@temple.edu.

The files ChatGPT_Initial.csv, ChatGPT_Variant1.csv, ChatGPT_Variant2.csv, ChatGPT\Variant3.csv, and ChatGPT\Variant4.csv 
are output from ChatGPT for the initial prompt and four variants in Sections 2.1--2.5 of the paper. They correspond to 
Tables 2, 4, 6, 8, and 10, respectively.

The files Claude_Initial.csv, Claude_Variant1.csv, Claude_Variant2.csv, Claude_Variant3.csv, Claude_Variant4.csv 
are corresponding output from Claude. The associated Tables are 3, 5, 7, 9, and 11.

The files ChatGPT_Initial_Temp0_v3_1000.csv, ChatGPT_initial_Temp0_v3_5000.csv, and ChatGPT_Initial_Temp0_v3_10000.csv 
contain the results of 16000 replications of a variant of the initial prompt. These results appear in 
Section 3.2 of the paper and underlie Figure 3.

All files were generated by Python scripts of the form shown in Appendix B of the paper.
